
# Early Stroke Detection using Voice and ML

This project detects early neurological disorders using speech analysis.

## Features Used
- MFCC
- Pitch
- Jitter, Shimmer
- HNR

## Models
- Logistic Regression
- SVM
- Random Forest
- XGBoost

## Results
Best model: XGBoost  
Accuracy: ~97%

Dataset not included due to size.
